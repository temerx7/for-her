📝 How to personalise this

1. Photos on background: In #bg-photos, replace each <div class="placeholder-icon"> with <img src="photos/photo1.jpg">.

2. Gallery strip: In #photoStrip, replace placeholder divs with <img src="photos/photo1.jpg"> inside each .photo-card.

3. Slideshow: In #slideshow, replace <div class="slide-placeholder"> with <img src="..."> or <video src="...">.

4. Love message: Edit the text inside #loveMessage.

5. Upload the index.html + a photos/ folder to GitHub and enable GitHub Pages in repo settings.
